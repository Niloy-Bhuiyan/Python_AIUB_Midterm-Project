1. Project Objective
The objective of this project is to develop a Pac-Man-style arcade game using Python and Pygame, demonstrating core programming concepts, Object-Oriented Programming (OOP), file handling, and use of external libraries.
2. Technologies Used
• Python 3
• Pygame Library
• JSON for data storage

3. Setup & Installation
1. Clone the repository
git clone https://github.com/Niloy-Bhuiyan/Python_AIUB_Midterm-Project.git

2. Navigate to project folder
cd Python_AIUB_Midterm-Project

3. Install dependencies (make sure Python 3 and pip are installed)
pip install pygame

4. Run the game
python pacman.py
